# STT 語音轉文字工具

這是一個簡單易用的語音轉文字（Speech To Text）網頁應用程式，讓使用者能夠輕鬆將音訊或影片檔案轉換成文字稿。

## 功能特點

- 🎯 支援多種音訊和影片格式
  - 音訊：MP3, WAV, OGG, AAC, M4A, FLAC, WMA
  - 影片：MP4, MOV, WebM, MKV

- 📝 自動生成內容摘要

- 💾 支援多種匯出格式
  - SRT 字幕檔
  - TXT 純文字檔

- 🔍 強大的查找替換功能
  - 關鍵字搜尋
  - 批量替換
  - 即時預覽

- 📱 響應式設計，支援各種裝置

## 使用限制

- 單一檔案大小上限：300MB
- 單一檔案時常上限：30分鐘
- 支援繁體中文介面
